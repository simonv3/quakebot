YourQuakeBot

Follow @yourquakebot to let it tell you when there's an earthquake near you.

YourQuakeBot is a twitter bot that follows USGIS magnitude 4.5 earthquake feeds and will send out a DM when an earthquake is detected to Twitter users following the bot within a 100km radius.

YourQuakeBot is an experiment and shouldn't be used for disaster preparation.

YourQuakeBot is open source - the code is hosted on GitHub. It's built by @simonv3.
